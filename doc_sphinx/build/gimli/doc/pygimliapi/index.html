pyGIMLi API Reference

pyGIMLi - An open-source library for modelling and inversion in geophysics

Module overview

frameworks

Unified and method independent inversion frameworks.

meshtools

Mesh generation and modification.

physics

Module containing submodules for various geophysical methods.

solver

General physics independent solver interface.

testing

Testing utilities

utils

Useful utility functions.

viewer

Interface for 2D and 3D visualizations.

Note

This documentation is valid for v0.1. Check your installed version with

import pygimli as pg
print(pg.__version__)

and consider updating (e.g., via the conda package manager). If you have a newer version, the current documentation of the dev branch before the next release is available at: https://dev.pygimli.org.
